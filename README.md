# User-Behavior-Engagement-Analysis-using-Power-BI

To analyze user behavior and engagement data for a comprehensive digital health and wellness platform, in order to uncover actionable insights that enhance user retention, feature adoption, and community participation. The goal is to develop interactive dashboards that visualize key performance indicators (KPIs), reveal usage trends, and support data-driven decision-making across core business units, ultimately driving improved user experiences and sustainable growth.
